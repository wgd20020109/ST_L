# ST语言（结构化文本）学习路线

## 概述

ST（Structured Text，结构化文本）是IEC 61131-3标准中定义的PLC编程语言之一，专为工业自动化控制系统设计。它类似于高级编程语言（如Pascal、C），采用文本方式进行编程，适合复杂的数学运算和算法实现。

## 预备知识

- 具备至少一门编程语言的基础（推荐C、Python或JavaScript）
- 了解基本的数据结构和算法概念
- 对工业自动化有基础认知（加分项，非必需）

## 学习目标

完成本系列教程后，你将能够：

1. 熟练使用ST语言编写PLC程序
2. 理解IEC 61131-3标准的核心概念
3. 设计并实现中等复杂度的自动化控制逻辑
4. 使用函数和功能块实现代码复用
5. 处理数组、结构等复杂数据类型

## 推荐开发环境

| 软件 | 特点 | 适用场景 |
|------|------|----------|
| CODESYS | 功能完整、支持多品牌PLC | 专业开发 |
| TwinCAT (Beckhoff) | 集成Visual Studio | 高级用户 |
| PLCopen XML Editor | 轻量级编辑器 | 学习研究 |
| Simulink PLC Coder | 仿真验证 | 算法开发 |

## 如何使用本教程

1. **按顺序学习**：建议按照编号顺序逐步深入
2. **动手实践**：每个知识点都配有代码示例，请务必在IDE中亲手编写运行
3. **理解原理**：不仅要知道"怎么写"，更要理解"为什么这样写"
4. **举一反三**：尝试修改示例代码，验证不同场景下的行为
