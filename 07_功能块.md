# 第7章：功能块（FUNCTION BLOCK）

## 7.1 功能块基础

### 什么是功能块？

功能块（FUNCTION BLOCK）是IEC 61131-3中最重要的代码组织单元，具有以下核心特点：

- **持久状态**：每次调用后内部状态保持
- **可实例化**：可以创建多个独立实例
- **封装性**：将数据和行为封装在一起
- **面向对象**：是PLC中面向对象编程的基础

### 功能块 vs 函数 vs 程序

| 特性 | 程序(PROGRAM) | 函数(FUNCTION) | 功能块(FB) |
|------|---------------|----------------|------------|
| 全局唯一 | ✅ | ✅ | ❌（可实例化） |
| 内部状态 | ❌ | ❌ | ✅ |
| 可创建实例 | ❌ | ❌ | ✅ |
| VAR_INPUT | ❌ | ✅ | ✅ |
| VAR_OUTPUT | ❌ | ✅ | ✅ |
| VAR | ❌ | ✅ | ✅ |
| 返回值 | ❌ | ✅ | ❌ |

## 7.2 功能块声明语法

### 基本语法结构

```st
FUNCTION_BLOCK <功能块名称>
VAR_INPUT
    (* 输入参数 - 外部传入 *)
END_VAR
VAR_OUTPUT
    (* 输出参数 - 输出到外部 *)
END_VAR
VAR_IN_OUT
    (* 输入输出参数 - 既输入又输出 *)
END_VAR
VAR
    (* 内部变量 - 保持状态 *)
END_VAR

BEGIN
    (* 功能块逻辑 *)
END_FUNCTION_BLOCK
```

### 简单功能块示例

```st
(* 计数器功能块 *)
FUNCTION_BLOCK Counter
VAR_INPUT
    enable : BOOL := TRUE;   (* 使能 *)
    preset : INT := 100;     (* 预设值 *)
    increment : BOOL;         (* 计数触发 *)
    decrement : BOOL;         (* 递减触发 *)
    reset : BOOL;             (* 复位 *)
END_VAR

VAR_OUTPUT
    currentValue : INT;       (* 当前值 *)
    done : BOOL;               (* 计数完成 *)
    overflow : BOOL;          (* 溢出标志 *)
END_VAR

VAR
    lastInc : BOOL := FALSE;
    lastDec : BOOL := FALSE;
    lastReset : BOOL := FALSE;
END_VAR

BEGIN
    (* 边沿检测 - 上升沿 *)
    IF increment AND NOT lastInc THEN
        IF currentValue < preset THEN
            currentValue := currentValue + 1;
        END_IF
    END_IF;

    (* 递减 *)
    IF decrement AND NOT lastDec THEN
        IF currentValue > 0 THEN
            currentValue := currentValue - 1;
        END_IF
    END_IF;

    (* 复位 *)
    IF reset AND NOT lastReset THEN
        currentValue := 0;
    END_IF;

    (* 状态输出 *)
    done := (currentValue >= preset);
    overflow := (currentValue >= 32767);  (* INT最大值附近 *)

    (* 保存上次状态 *)
    lastInc := increment;
    lastDec := decrement;
    lastReset := reset;
END_FUNCTION_BLOCK
```

## 7.3 功能块实例化

### 创建实例

在程序或全局变量中声明功能块实例：

```st
PROGRAM MainProgram
VAR
    (* 创建功能块实例 *)
    motor1Counter : Counter;
    motor2Counter : Counter;
    sensor1Counter : Counter;

    (* 其他变量 *)
    motorRunning : BOOL;
    sensorTriggered : BOOL;
END_VAR

BEGIN
    (* 调用功能块 *)
    motor1Counter(
        enable := TRUE,
        preset := 1000,
        increment := motorRunning,
        decrement := FALSE,
        reset := FALSE,
        currentValue => motor1Count,
        done => motor1Done,
        overflow => motor1Overflow
    );

    sensor1Counter(
        enable := TRUE,
        preset := 100,
        increment := sensorTriggered,
        decrement := FALSE,
        reset := systemReset,
        currentValue => sensor1Count,
        done => sensor1Done,
        overflow =>
    );
END_PROGRAM
```

### 实例独立性验证

```st
PROGRAM InstanceDemo
VAR
    (* 两个独立的计数器实例 *)
    counterA : Counter;
    counterB : Counter;

    (* 输入信号 *)
    pulseA : BOOL;
    pulseB : BOOL;
END_VAR

BEGIN
    (* 每个实例有独立的状态 *)
    counterA(preset := 10, increment := pulseA);
    counterB(preset := 20, increment := pulseB);

    (* counterA.currentValue 和 counterB.currentValue 是独立的 *)
    (* 即使preset设置相同，currentValue也会各自独立计数 *)
END_PROGRAM
```

## 7.4 定时器功能块

### TON - 通电延时定时器

```st
(*
 * TON (Timer On-Delay)
 * 特性：输入为TRUE时，延时PT后输出Q变为TRUE
 *       输入变为FALSE时，输出立即变为FALSE
 *)

FUNCTION_BLOCK TON
VAR_INPUT
    IN : BOOL;    (* 触发输入 *)
    PT : TIME;    (* 延时时间 *)
END_VAR
VAR_OUTPUT
    Q : BOOL;     (* 输出 *)
    ET : TIME;    (* 已计时时间 *)
END_VAR
VAR
    startTime : TIME;
    running : BOOL := FALSE;
END_VAR

BEGIN
    IF IN AND NOT running THEN
        (* 启动定时 *)
        running := TRUE;
        startTime := TIME();
        Q := FALSE;
    ELSIF NOT IN THEN
        (* 停止并复位 *)
        running := FALSE;
        Q := FALSE;
        ET := T#0s;
    ELSIF running THEN
        (* 检查是否到达延时时间 *)
        ET := TIME() - startTime;
        IF ET >= PT THEN
            Q := TRUE;
            ET := PT;
        END_IF
    END_IF
END_FUNCTION_BLOCK
```

### TOF - 断电延时定时器

```st
(*
 * TOF (Timer Off-Delay)
 * 特性：输入变为FALSE时，延时PT后输出Q变为FALSE
 *       输入变为TRUE时，输出立即变为TRUE
 *)

FUNCTION_BLOCK TOF
VAR_INPUT
    IN : BOOL;    (* 触发输入 *)
    PT : TIME;    (* 延时时间 *)
END_VAR
VAR_OUTPUT
    Q : BOOL;     (* 输出 *)
    ET : TIME;    (* 已计时时间 *)
END_VAR
VAR
    lastIN : BOOL := FALSE;
    startTime : TIME;
    running : BOOL := FALSE;
END_VAR

BEGIN
    IF IN AND NOT lastIN THEN
        (* 输入变为TRUE，立即输出TRUE *)
        Q := TRUE;
        running := FALSE;
    ELSIF NOT IN AND lastIN THEN
        (* 输入变为FALSE，启动延时 *)
        running := TRUE;
        startTime := TIME();
    ELSIF running THEN
        ET := TIME() - startTime;
        IF ET >= PT THEN
            Q := FALSE;
            running := FALSE;
            ET := T#0s;
        END_IF
    END_IF

    lastIN := IN;
END_FUNCTION_BLOCK
```

### TP - 脉冲定时器

```st
(*
 * TP (Timer Pulse)
 * 特性：每次触发产生固定宽度的脉冲
 *)

FUNCTION_BLOCK TP
VAR_INPUT
    IN : BOOL;    (* 触发输入 *)
    PT : TIME;    (* 脉冲宽度 *)
END_VAR
VAR_OUTPUT
    Q : BOOL;     (* 脉冲输出 *)
    ET : TIME;    (* 已计时时间 *)
END_VAR
VAR
    lastIN : BOOL := FALSE;
    startTime : TIME;
    running : BOOL := FALSE;
END_VAR

BEGIN
    (* 检测上升沿 *)
    IF IN AND NOT lastIN AND NOT running THEN
        running := TRUE;
        startTime := TIME();
        Q := TRUE;
    END_IF;

    (* 计时 *)
    IF running THEN
        ET := TIME() - startTime;
        IF ET >= PT THEN
            Q := FALSE;
            running := FALSE;
            ET := PT;
        END_IF
    ELSE
        ET := T#0s;
    END_IF;

    lastIN := IN;
END_FUNCTION_BLOCK
```

### 定时器使用示例

```st
PROGRAM TimerDemo
VAR
    startButton : BOOL;
    stopButton : BOOL;
    conveyorMotor : BOOL;

    (* 定时器实例 *)
    startDelay : TON;
    stopDelay : TOF;
    warningFlash : TP;
END_VAR

BEGIN
    (* 启动延时：按下按钮3秒后启动 *)
    startDelay(IN := startButton, PT := T#3s);
    conveyorMotor := startDelay.Q;

    (* 停止延时：松开按钮5秒后停止 *)
    stopDelay(IN := stopButton, PT := T#5s);

    (* 故障闪烁：每500ms闪烁一次 *)
    warningFlash(IN := faultPresent, PT := T#500ms);
    faultIndicator := warningFlash.Q;
END_PROGRAM
```

## 7.5 边沿检测功能块

### R_TRIG - 上升沿检测

```st
(*
 * R_TRIG (Rising Edge Trigger)
 * 检测输入从FALSE变为TRUE的时刻
 *)

FUNCTION_BLOCK R_TRIG
VAR_INPUT
    CLK : BOOL;    (* 输入信号 *)
END_VAR
VAR_OUTPUT
    Q : BOOL;     (* 上升沿脉冲 *)
END_VAR
VAR
    lastCLK : BOOL := FALSE;
END_VAR

BEGIN
    Q := CLK AND NOT lastCLK;  (* 检测上升沿 *)
    lastCLK := CLK;
END_FUNCTION_BLOCK
```

### F_TRIG - 下降沿检测

```st
(*
 * F_TRIG (Falling Edge Trigger)
 * 检测输入从TRUE变为FALSE的时刻
 *)

FUNCTION_BLOCK F_TRIG
VAR_INPUT
    CLK : BOOL;    (* 输入信号 *)
END_VAR
VAR_OUTPUT
    Q : BOOL;     (* 下降沿脉冲 *)
END_VAR
VAR
    lastCLK : BOOL := FALSE;
END_VAR

BEGIN
    Q := NOT CLK AND lastCLK;  (* 检测下降沿 *)
    lastCLK := CLK;
END_FUNCTION_BLOCK
```

### 边沿检测使用示例

```st
PROGRAM EdgeDetectionDemo
VAR
    sensor : BOOL;
    startSignal : BOOL;
    stopSignal : BOOL;

    (* 边沿检测实例 *)
    sensorRising : R_TRIG;
    startRising : R_TRIG;
    stopRising : R_TRIG;

    (* 计数器 *)
    partCount : INT := 0;
END_VAR

BEGIN
    (* 检测传感器上升沿，计数 *)
    sensorRising(CLK := sensor);
    IF sensorRising.Q THEN
        partCount := partCount + 1;
    END_IF;

    (* 启动按钮上升沿触发 *)
    startRising(CLK := startSignal);
    IF startRising.Q THEN
        motorRunning := TRUE;
    END_IF;

    (* 停止按钮上升沿触发 *)
    stopRising(CLK := stopSignal);
    IF stopRising.Q THEN
        motorRunning := FALSE;
        partCount := 0;  (* 复位计数 *)
    END_IF
END_PROGRAM
```

## 7.6 实际应用案例

### 案例1：电机控制功能块

```st
(*
 * 电机控制功能块
 * 包含启动/停止、速度控制、故障检测
 *)

FUNCTION_BLOCK MotorControl
VAR_INPUT
    enable : BOOL;              (* 使能 *)
    start : BOOL;               (* 启动命令 *)
    stop : BOOL;                (* 停止命令 *)
    faultReset : BOOL;          (* 故障复位 *)
    actualSpeed : INT;          (* 实际转速 *)
    overcurrent : BOOL;         (* 过流信号 *)
    overtemperature : BOOL;     (* 过温信号 *)
END_VAR

VAR_OUTPUT
    runCommand : BOOL;           (* 运行命令输出 *)
    speedSetpoint : INT;        (* 速度设定 *)
    ready : BOOL;               (* 就绪状态 *)
    running : BOOL;             (* 运行状态 *)
    fault : BOOL;               (* 故障状态 *)
    faultCode : INT;            (* 故障代码 *)
END_VAR

VAR
    startDelay : TON;
    stopDelay : TON;
    faultDelay : TON;
    state : INT := 0;           (* 0=停止, 1=启动中, 2=运行, 3=故障 *)
END_VAR

BEGIN
    (* 状态机逻辑 *)
    CASE state OF
        0: (* 停止状态 *)
            ready := TRUE;
            running := FALSE;
            runCommand := FALSE;

            IF enable AND start THEN
                state := 1;     (* 转入启动 *)
            END_IF

        1: (* 启动中 *)
            ready := FALSE;
            startDelay(IN := TRUE, PT := T#2s);

            IF startDelay.Q THEN
                startDelay(IN := FALSE);
                runCommand := TRUE;
                state := 2;    (* 转入运行 *)
            ELSIF fault THEN
                state := 3;    (* 故障 *)
            END_IF

        2: (* 运行状态 *)
            ready := TRUE;
            running := TRUE;
            speedSetpoint := 1500;

            (* 停止命令 *)
            IF stop THEN
                runCommand := FALSE;
                state := 0;
            (* 故障检测 *)
            ELSIF overcurrent THEN
                faultCode := 1;
                state := 3;
            ELSIF overtemperature THEN
                faultCode := 2;
                state := 3;
            END_IF

        3: (* 故障状态 *)
            ready := FALSE;
            running := FALSE;
            runCommand := FALSE;

            IF faultReset THEN
                faultCode := 0;
                state := 0;
            END_IF
    END_CASE
END_FUNCTION_BLOCK
```

### 案例2：液位控制功能块

```st
(*
 * 液位控制功能块
 * 使用PID算法控制液位
 *)

FUNCTION_BLOCK LevelControl
VAR_INPUT
    enable : BOOL;
    setpoint : REAL;             (* 液位设定值 *)
    actualLevel : REAL;         (* 实际液位 *)
    fillValve : BOOL;           (* 进水阀 *)
    drainValve : BOOL;          (* 排水阀 *)
END_VAR

VAR_OUTPUT
    fillCommand : BOOL;        (* 进水命令 *)
    drainCommand : BOOL;        (* 排水命令 *)
    levelStatus : INT;         (* 0=正常, 1=低报警, 2=高报警 *)
END_VAR

VAR
    kp : REAL := 2.0;
    ki : REAL := 0.5;
    integral : REAL := 0.0;
    lastError : REAL := 0.0;
    error : REAL;

    lowLimit : REAL := 10.0;
    highLimit : REAL := 90.0;
    lowAlarm : REAL := 20.0;
    highAlarm : REAL := 80.0;
END_VAR

BEGIN
    IF enable THEN
        (* 计算误差 *)
        error := setpoint - actualLevel;

        (* PID积分 *)
        integral := integral + ki * error * 0.1;  (* 100ms采样 *)
        IF integral > 50.0 THEN integral := 50.0; END_IF
        IF integral < -50.0 THEN integral := -50.0; END_IF

        (* 液位报警判断 *)
        IF actualLevel <= lowAlarm THEN
            levelStatus := 1;   (* 低报警 *)
        ELSIF actualLevel >= highAlarm THEN
            levelStatus := 2;   (* 高报警 *)
        ELSE
            levelStatus := 0;   (* 正常 *)
        END_IF

        (* 简单液位控制 *)
        IF actualLevel < (setpoint - 5.0) THEN
            fillCommand := TRUE;
            drainCommand := FALSE;
        ELSIF actualLevel > (setpoint + 5.0) THEN
            fillCommand := FALSE;
            drainCommand := TRUE;
        ELSE
            fillCommand := FALSE;
            drainCommand := FALSE;
        END_IF

        lastError := error;
    ELSE
        (* 禁用状态 *)
        fillCommand := FALSE;
        drainCommand := FALSE;
        levelStatus := 0;
        integral := 0.0;
    END_IF
END_FUNCTION_BLOCK
```

### 案例3：批处理控制功能块

```st
(*
 * 批处理配方功能块
 * 按照配方顺序执行各步骤
 *)

TYPE
    BatchState : (
        BATCH_IDLE,
        BATCH_FILLING,
        BATCH_HEATING,
        BATCH_MIXING,
        BATCH_DRAINING,
        BATCH_COMPLETE,
        BATCH_ABORTED
    );
END_TYPE

FUNCTION_BLOCK BatchController
VAR_INPUT
    start : BOOL;
    abort : BOOL;
    tankLevel : REAL;           (* 罐体液位 *)
    temperature : REAL;         (* 温度 *)
    tankFull : BOOL;
    tankEmpty : BOOL;
    temperatureReached : BOOL;
    mixingComplete : BOOL;
END_VAR

VAR_OUTPUT
    fillValve : BOOL;
    drainValve : BOOL;
    heater : BOOL;
    mixer : BOOL;
    currentState : BatchState;
    progress : INT;             (* 进度百分比 *)
END_VAR

VAR
    state : BatchState := BATCH_IDLE;
    timer : TON;
    progressTimer : TON;
END_VAR

BEGIN
    (* 状态机主逻辑 *)
    CASE state OF
        BATCH_IDLE:
            fillValve := FALSE;
            drainValve := FALSE;
            heater := FALSE;
            mixer := FALSE;
            progress := 0;

            IF start THEN
                state := BATCH_FILLING;
            END_IF

        BATCH_FILLING:
            fillValve := TRUE;
            heater := FALSE;
            mixer := FALSE;

            IF tankFull THEN
                fillValve := FALSE;
                state := BATCH_HEATING;
            ELSIF abort THEN
                state := BATCH_ABORTED;
            END_IF

        BATCH_HEATING:
            fillValve := FALSE;
            heater := TRUE;
            mixer := FALSE;

            IF temperatureReached THEN
                heater := FALSE;
                state := BATCH_MIXING;
            ELSIF abort THEN
                heater := FALSE;
                state := BATCH_ABORTED;
            END_IF

        BATCH_MIXING:
            fillValve := FALSE;
            heater := FALSE;
            mixer := TRUE;

            IF mixingComplete THEN
                mixer := FALSE;
                state := BATCH_DRAINING;
            ELSIF abort THEN
                mixer := FALSE;
                state := BATCH_ABORTED;
            END_IF

        BATCH_DRAINING:
            drainValve := TRUE;

            IF tankEmpty THEN
                drainValve := FALSE;
                state := BATCH_COMPLETE;
            ELSIF abort THEN
                state := BATCH_ABORTED;
            END_IF

        BATCH_COMPLETE:
            drainValve := FALSE;
            progress := 100;
            IF start THEN
                state := BATCH_IDLE;  (* 准备下一批 *)
            END_IF

        BATCH_ABORTED:
            fillValve := FALSE;
            drainValve := TRUE;
            heater := FALSE;
            mixer := FALSE;

            IF tankEmpty THEN
                drainValve := FALSE;
                state := BATCH_IDLE;
            END_IF
    END_CASE;

    (* 进度计算 *)
    CASE state OF
        BATCH_IDLE:       progress := 0;
        BATCH_FILLING:    progress := 20;
        BATCH_HEATING:    progress := 50;
        BATCH_MIXING:     progress := 75;
        BATCH_DRAINING:   progress := 90;
        BATCH_COMPLETE:   progress := 100;
        BATCH_ABORTED:    progress := 0;
    END_CASE;

    currentState := state;
END_FUNCTION_BLOCK
```

## 7.7 功能块的最佳实践

### 初始化内部状态

```st
FUNCTION_BLOCK WellDesignedFB
VAR_INPUT
    enable : BOOL;
    initialValue : INT := 0;
END_VAR
VAR_OUTPUT
    currentValue : INT;
END_VAR
VAR
    firstCycle : BOOL := TRUE;  (* 标记首次执行 *)
    internalState : INT;
END_VAR

BEGIN
    (* 首次调用时初始化 *)
    IF firstCycle THEN
        internalState := initialValue;
        firstCycle := FALSE;
    END_IF;

    (* 正常逻辑 *)
    IF enable THEN
        internalState := internalState + 1;
    END_IF;

    currentValue := internalState;
END_FUNCTION_BLOCK
```

### 命名规范

```st
(* 推荐：清晰的功能块和变量命名 *)
FUNCTION_BLOCK BottleFilling
VAR_INPUT
    startFilling : BOOL;
    bottlePresent : BOOL;
    levelSensor : BOOL;
END_VAR

VAR_OUTPUT
    valveOpen : BOOL;
    fillComplete : BOOL;
END_VAR

(* 不推荐：含糊不清的命名 *)
FUNCTION_BLOCK FB1  (* 名称无意义 *)
VAR_INPUT
    in1, in2 : BOOL;
END_VAR
```

### 单一职责原则

```st
(* 推荐：功能单一 *)
FUNCTION_BLOCK ValveControl     (* 只控制阀门 *)
FUNCTION_BLOCK LevelMeasurement  (* 只测量液位 *)
FUNCTION_BLOCK TemperatureControl (* 只控制温度 *)

(* 不推荐：功能混杂 *)
FUNCTION_BLOCK TankControl
    (* 同时处理液位、温度、压力、搅拌... *)
```

## 7.8 功能块调用方法

### 方法1：函数式调用

```st
(* 直接调用，返回值到输出参数 *)
counter(
    enable := TRUE,
    preset := 100,
    increment := pulse,
    currentValue => count,
    done =>
);
```

### 方法2：分离输入输出

```st
(* 为输入赋值 *)
counter.enable := TRUE;
counter.preset := 100;
counter.increment := pulse;

(* 在适当位置调用功能块 *)
counter();

(* 读取输出 *)
count := counter.currentValue;
doneFlag := counter.done;
```

## 7.9 本章小结

### 功能块核心概念

| 概念 | 说明 |
|------|------|
| 实例化 | 功能块可以创建多个独立实例 |
| 持久状态 | 内部变量在调用间保持 |
| VAR_INPUT | 输入参数 |
| VAR_OUTPUT | 输出参数 |
| VAR | 内部状态变量 |

### 常用标准功能块

| 功能块 | 用途 |
|--------|------|
| TON | 通电延时定时器 |
| TOF | 断电延时定时器 |
| TP | 脉冲定时器 |
| R_TRIG | 上升沿检测 |
| F_TRIG | 下降沿检测 |
| CTU | 计数（递增） |
| CTD | 计数（递减） |
| CTUD | 双向计数器 |

### 实践任务

1. 创建一个RS触发器功能块（置位优先或复位优先）
2. 实现一个单按钮启停功能块
3. 创建一个生成PWM脉冲的功能块
4. 实现一个带超时的阀门控制功能块
5. 创建一个多段温度曲线控制功能块
