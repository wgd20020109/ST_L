# 第6章：函数（FUNCTION）

## 6.1 函数基础

### 什么是函数？

函数（FUNCTION）是IEC 61131-3中的代码封装单元，具有以下特点：
- **无持久状态**：每次调用都是独立的
- **单一返回值**：通过函数名返回结果
- **可嵌套调用**：函数可以调用其他函数
- **可重复使用**：相同逻辑只需编写一次

### 函数与程序的区别

| 特性 | 程序(PROGRAM) | 函数(FUNCTION) |
|------|---------------|----------------|
| 可创建实例 | 否（全局） | 否（无状态） |
| 可多次调用 | 循环扫描 | 任意位置 |
| 内部状态 | 无 | 无 |
| 返回值 | 无 | 有（必须） |
| VAR_INPUT | 否 | 是 |
| VAR_OUTPUT | 否 | 是 |
| VAR_IN_OUT | 否 | 是 |

## 6.2 函数声明语法

### 基本语法结构

```st
FUNCTION <函数名> : <返回类型>
VAR_INPUT
    (* 输入参数声明 *)
END_VAR
VAR_OUTPUT
    (* 输出参数声明 *)
END_VAR
VAR_IN_OUT
    (* 输入输出参数声明 *)
END_VAR
VAR
    (* 局部变量声明 *)
END_VAR

(* 函数体 *)
BEGIN
    (* 执行逻辑 *)
    <函数名> := <返回值>;  (* 赋值给函数名作为返回值 *)
END_FUNCTION
```

### 最简单的函数

```st
(* 无参数函数 *)
FUNCTION AddOne : INT
VAR
    counter : INT := 0;
END_VAR

BEGIN
    counter := counter + 1;
    AddOne := counter;  (* 返回counter *)
END_FUNCTION

(* 等价于：*)
FUNCTION AddOneSimple : INT
BEGIN
    AddOneSimple := 1;
END_FUNCTION
```

## 6.3 参数类型详解

### VAR_INPUT - 输入参数

```st
(* 计算两数之和 *)
FUNCTION Add : INT
VAR_INPUT
    a : INT;
    b : INT;
END_VAR

BEGIN
    Add := a + b;
END_FUNCTION

(* 使用示例 *)
result := Add(10, 20);  (* result = 30 *)
```

### VAR_OUTPUT - 输出参数

```st
(* 分离整数和小数部分 *)
FUNCTION SplitNumber
VAR_INPUT
    input : REAL;
END_VAR
VAR_OUTPUT
    integerPart : INT;
    fractionPart : REAL;
END_VAR

BEGIN
    integerPart := REAL_TO_INT(input);
    fractionPart := input - INT_TO_REAL(integerPart);
END_FUNCTION

(* 使用示例 *)
SplitNumber(input := 3.14159, integerPart => intPart, fractionPart => frac);
(* intPart = 3, frac = 0.14159 *)
```

### VAR_IN_OUT - 输入输出参数

```st
(* 交换两个数的值 *)
FUNCTION Swap
VAR_IN_OUT
    a : INT;
    b : INT;
END_VAR
VAR
    temp : INT;
END_VAR

BEGIN
    temp := a;
    a := b;
    b := temp;
END_FUNCTION

(* 使用示例 *)
x := 10; y := 20;
Swap(a := x, b := y);  (* x = 20, y = 10 *)
```

### 参数修饰符

| 修饰符 | 说明 | 用途 |
|--------|------|------|
| (无) | 按值传递 | 输入参数 |
| VAR_INPUT | 按值传递 | 输入参数 |
| VAR_OUTPUT | 按值返回 | 输出参数 |
| VAR_IN_OUT | 按引用传递 | 输入输出参数 |

## 6.4 标准数学函数

IEC 61131-3标准库提供常用数学函数：

```st
VAR
    x, y, result : REAL;
    a, b, angle : REAL;
END_VAR

(* 绝对值 *)
result := ABS(-5.5);              (* 5.5 *)
result := ABS(3);                 (* 3 *)

(* 平方根 *)
result := SQRT(16.0);             (* 4.0 *)

(* 幂运算 *)
result := EXPT(2.0, 3);           (* 8.0, 2^3 *)
result := POW(2.0, 3);           (* 8.0, 同上 *)

(* 对数 *)
result := LN(2.71828);            (* 自然对数 *)
result := LOG(100.0);            (* 常用对数, 2.0 *)

(* 三角函数（弧度） *)
angle := 3.14159 / 2;             (* 90度 *)
result := SIN(angle);             (* 1.0 *)
result := COS(angle);             (* 0.0 *)
result := TAN(angle);             (* 很大值 *)

(* 反三角函数 *)
result := ASIN(1.0);              (* π/2 *)
result := ACOS(0.0);              (* π/2 *)
result := ATAN(1.0);              (* π/4 *)

(* 角度转换 *)
result := SIN(DTOR(angle));       (* 角度转弧度后计算 *)
result := RAD_TO_DEG(ATAN(1.0)); (* 弧度转角度, 45.0 *)

(* 取整函数 *)
result := TRUNC(3.7);            (* 3.0, 截断 *)
result := ROUND(3.7);            (* 4.0, 四舍五入 *)
result := FLOOR(3.7);           (* 3.0, 向下取整 *)
result := CEIL(3.2);             (* 4.0, 向上取整 *)

(* 最大最小值 *)
result := MIN(a, b, x, y);
result := MAX(a, b, x, y);
result := LIMIT(0.0, x, 100.0); (* 限幅 *)
```

## 6.5 函数重载

### 隐式函数重载

某些函数可以接受不同类型的参数：

```st
VAR
    intVal : INT := 5;
    realVal : REAL := 3.14;
    absInt : INT;
    absReal : REAL;
END_VAR

(* ABS函数可接受多种类型 *)
absInt := ABS(intVal);     (* INT版本 *)
absReal := ABS(realVal);   (* REAL版本 *)
```

## 6.6 实战案例

### 案例1：温度单位转换

```st
(* 摄氏度转华氏度 *)
FUNCTION CelsiusToFahrenheit : REAL
VAR_INPUT
    celsius : REAL;
END_VAR

BEGIN
    CelsiusToFahrenheit := celsius * 9.0 / 5.0 + 32.0;
END_FUNCTION

(* 华氏度转摄氏度 *)
FUNCTION FahrenheitToCelsius : REAL
VAR_INPUT
    fahrenheit : REAL;
END_VAR

BEGIN
    FahrenheitToCelsius := (fahrenheit - 32.0) * 5.0 / 9.0;
END_FUNCTION

(* 摄氏度转开尔文 *)
FUNCTION CelsiusToKelvin : REAL
VAR_INPUT
    celsius : REAL;
END_VAR

BEGIN
    CelsiusToKelvin := celsius + 273.15;
END_FUNCTION
```

### 案例2：几何计算函数库

```st
(* 计算圆的面积 *)
FUNCTION CircleArea : REAL
VAR_INPUT
    radius : REAL;
END_VAR

BEGIN
    IF radius < 0.0 THEN
        CircleArea := 0.0;
    ELSE
        CircleArea := 3.14159 * radius * radius;
    END_IF
END_FUNCTION

(* 计算圆的周长 *)
FUNCTION CircleCircumference : REAL
VAR_INPUT
    radius : REAL;
END_VAR

BEGIN
    CircleCircumference := 2.0 * 3.14159 * radius;
END_FUNCTION

(* 计算矩形面积 *)
FUNCTION RectangleArea : REAL
VAR_INPUT
    width : REAL;
    height : REAL;
END_VAR

BEGIN
    RectangleArea := ABS(width) * ABS(height);
END_FUNCTION

(* 计算三角形面积（海伦公式） *)
FUNCTION TriangleArea : REAL
VAR_INPUT
    a, b, c : REAL;  (* 三边长度 *)
END_VAR
VAR
    s : REAL;  (* 半周长 *)
END_VAR

BEGIN
    (* 三角形存在条件：任意两边之和大于第三边 *)
    IF (a + b > c) AND (b + c > a) AND (a + c > b) THEN
        s := (a + b + c) / 2.0;
        TriangleArea := SQRT(s * (s - a) * (s - b) * (s - c));
    ELSE
        TriangleArea := 0.0;  (* 无效三角形 *)
    END_IF
END_FUNCTION

(* 计算两点之间的距离 *)
FUNCTION Distance2D : REAL
VAR_INPUT
    x1, y1 : REAL;
    x2, y2 : REAL;
END_VAR

BEGIN
    Distance2D := SQRT(EXPT(x2 - x1, 2) + EXPT(y2 - y1, 2));
END_FUNCTION
```

### 案例3：数据处理函数

```st
(* 计算数组平均值 *)
FUNCTION ArrayAverage : REAL
VAR_INPUT
    data : ARRAY[1..100] OF INT;
    count : INT;
END_VAR
VAR
    i : INT;
    sum : DINT;
END_VAR

BEGIN
    IF count <= 0 OR count > 100 THEN
        ArrayAverage := 0.0;
        RETURN;
    END_IF;

    sum := 0;
    FOR i := 1 TO count DO
        sum := sum + data[i];
    END_FOR;

    ArrayAverage := DINT_TO_REAL(sum) / INT_TO_REAL(count);
END_FUNCTION

(* 计算数组标准差 *)
FUNCTION ArrayStdDev : REAL
VAR_INPUT
    data : ARRAY[1..100] OF INT;
    count : INT;
END_VAR
VAR
    i : INT;
    mean : REAL;
    variance : REAL;
    sumSq : LREAL;
END_VAR

BEGIN
    IF count <= 1 THEN
        ArrayStdDev := 0.0;
        RETURN;
    END_IF;

    (* 计算平均值 *)
    mean := ArrayAverage(data, count);

    (* 计算方差 *)
    sumSq := 0.0;
    FOR i := 1 TO count DO
        sumSq := sumSq + EXPT(INT_TO_REAL(data[i]) - mean, 2);
    END_FOR;
    variance := LREAL_TO_REAL(sumSq) / INT_TO_REAL(count);

    (* 返回标准差 *)
    ArrayStdDev := SQRT(variance);
END_FUNCTION

(* 计算数组最大值 *)
FUNCTION ArrayMax : INT
VAR_INPUT
    data : ARRAY[1..100] OF INT;
    count : INT;
END_VAR
VAR
    i : INT;
    maxVal : INT;
END_VAR

BEGIN
    IF count <= 0 OR count > 100 THEN
        ArrayMax := 0;
        RETURN;
    END_IF;

    maxVal := data[1];
    FOR i := 2 TO count DO
        IF data[i] > maxVal THEN
            maxVal := data[i];
        END_IF;
    END_FOR;

    ArrayMax := maxVal;
END_FUNCTION
```

### 案例4：PID参数计算

```st
(* 根据给定参数计算PID输出 *)
FUNCTION CalculatePID : REAL
VAR_INPUT
    setpoint : REAL;        (* 设定值 *)
    processValue : REAL;    (* 过程值 *)
    kp : REAL;              (* 比例系数 *)
    ki : REAL;              (* 积分系数 *)
    kd : REAL;              (* 微分系数 *)
    dt : REAL;              (* 采样时间 *)
END_VAR
VAR
    error : REAL;
    prevError : REAL := 0.0;
    integral : REAL := 0.0;
    derivative : REAL;
    output : REAL;
END_VAR

BEGIN
    (* 计算当前误差 *)
    error := setpoint - processValue;

    (* 积分项 *)
    integral := integral + error * dt;

    (* 防止积分饱和 *)
    IF integral > 100.0 THEN integral := 100.0; END_IF
    IF integral < -100.0 THEN integral := -100.0; END_IF

    (* 微分项 *)
    IF dt > 0.0 THEN
        derivative := (error - prevError) / dt;
    ELSE
        derivative := 0.0;
    END_IF

    (* PID计算 *)
    output := kp * error + ki * integral + kd * derivative;

    (* 输出限幅 *)
    IF output > 100.0 THEN output := 100.0; END_IF
    IF output < -100.0 THEN output := -100.0; END_IF

    prevError := error;
    CalculatePID := output;
END_FUNCTION

(* 计算PID参数的Ziegler-Nichols参数 *)
TYPE
    PIDTuning : STRUCT
        kp : REAL;
        ti : REAL;  (* 积分时间 *)
        td : REAL;  (* 微分时间 *)
    END_STRUCT
END_TYPE

FUNCTION CalculateZieglerNichols : PIDTuning
VAR_INPUT
    ku : REAL;    (* 临界增益 *)
    tu : REAL;    (* 临界周期 *)
END_VAR
VAR
    result : PIDTuning;
END_VAR

BEGIN
    (* P控制器 *)
    result.kp := 0.5 * ku;
    result.ti := 0.0;
    result.td := 0.0;

    (* PI控制器 *)
    result.kp := 0.45 * ku;
    result.ti := tu / 1.2;
    result.td := 0.0;

    (* PID控制器 *)
    result.kp := 0.6 * ku;
    result.ti := tu / 2.0;
    result.td := tu / 8.0;

    CalculateZieglerNichols := result;
END_FUNCTION
```

### 案例5：字符串处理函数

```st
(* 连接两个字符串 *)
FUNCTION ConcatStrings : STRING(100)
VAR_INPUT
    str1 : STRING(50);
    str2 : STRING(50);
END_VAR

BEGIN
    ConcatStrings := CONCAT(str1, str2);
END_FUNCTION

(* 获取字符串长度（简化实现） *)
FUNCTION StringLength : INT
VAR_INPUT
    str : STRING(100);
END_VAR
VAR
    i : INT;
END_VAR

BEGIN
    StringLength := 0;
    FOR i := 1 TO 100 DO
        IF MID(str, 1, i) <> '' THEN
            StringLength := i;
        END_IF;
    END_FOR;
END_FUNCTION

(* 字符串是否相等（不区分大小写） *)
FUNCTION StringsEqualIgnoreCase : BOOL
VAR_INPUT
    str1 : STRING(50);
    str2 : STRING(50);
END_VAR

BEGIN
    StringsEqualIgnoreCase := (UPPER(str1) = UPPER(str2));
END_FUNCTION
```

## 6.7 递归函数

ST语言支持函数递归调用：

```st
(* 递归计算阶乘 *)
FUNCTION Factorial : DINT
VAR_INPUT
    n : INT;
END_VAR

BEGIN
    IF n <= 1 THEN
        Factorial := 1;
    ELSE
        Factorial := INT_TO_DINT(n) * Factorial(n - 1);
    END_IF;
END_FUNCTION

(* 递归计算斐波那契数列（低效但直观） *)
FUNCTION FibonacciRecursive : DINT
VAR_INPUT
    n : INT;
END_VAR

BEGIN
    IF n <= 1 THEN
        FibonacciRecursive := INT_TO_DINT(n);
    ELSE
        FibonacciRecursive := FibonacciRecursive(n - 1) + FibonacciRecursive(n - 2);
    END_IF;
END_FUNCTION

(* 递归计算最大公约数（欧几里得算法） *)
FUNCTION GCD : DINT
VAR_INPUT
    a : DINT;
    b : DINT;
END_VAR

BEGIN
    IF b = 0 THEN
        GCD := a;
    ELSE
        GCD := GCD(b, a MOD b);
    END_IF;
END_FUNCTION
```

## 6.8 函数设计最佳实践

### 单一职责原则

```st
(* 推荐：单一职责 *)
FUNCTION CalculateNetPrice : REAL
VAR_INPUT
    price : REAL;
    taxRate : REAL;
    discountRate : REAL;
END_VAR

BEGIN
    CalculateNetPrice := price * (1 - discountRate) * (1 + taxRate);
END_FUNCTION

(* 不推荐：多个职责混合 *)
(* 计算价格、检查库存、生成发票... 这样的函数应该拆分 *)
```

### 边界检查

```st
FUNCTION SafeDivide : REAL
VAR_INPUT
    dividend : REAL;
    divisor : REAL;
END_VAR

BEGIN
    IF divisor = 0.0 THEN
        SafeDivide := 0.0;  (* 或其他错误处理 *)
    ELSE
        SafeDivide := dividend / divisor;
    END_IF
END_FUNCTION
```

### 文档注释

```st
(*
 * 函数: CalculateBMI
 * 描述: 计算身体质量指数
 * 输入: height - 身高（厘米）
 *       weight - 体重（公斤）
 * 返回: BMI值
 * 说明: BMI = 体重(kg) / 身高(m)^2
 *       < 18.5: 过轻
 *       18.5-24: 正常
 *       > 24: 过重
 *)
FUNCTION CalculateBMI : REAL
VAR_INPUT
    height : REAL;  (* 身高，单位：厘米 *)
    weight : REAL;  (* 体重，单位：公斤 *)
END_VAR

BEGIN
    IF height <= 0.0 OR weight <= 0.0 THEN
        CalculateBMI := 0.0;
        RETURN;
    END_IF;

    CalculateBMI := weight / EXPT(height / 100.0, 2);
END_FUNCTION
```

## 6.9 本章小结

### 函数声明要素

| 要素 | 说明 | 示例 |
|------|------|------|
| 函数名 | 标识符 | `FUNCTION Add` |
| 返回类型 | 函数返回值类型 | `: INT`, `: REAL` |
| VAR_INPUT | 输入参数 | `a : INT` |
| VAR_OUTPUT | 输出参数 | `result : INT` |
| VAR_IN_OUT | 输入输出参数 | `value : INT` |
| 返回值赋值 | 通过函数名赋值 | `Add := a + b` |

### 函数 vs 功能块

| 特性 | 函数 | 功能块 |
|------|------|--------|
| 状态保持 | 否 | 是 |
| 可多次实例化 | 否 | 是 |
| 内存占用 | 低 | 高（每个实例） |
| 执行效率 | 高 | 中等 |
| 适用场景 | 纯计算 | 有状态逻辑 |
