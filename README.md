# ST_L

本仓库包含使用 Smalltalk 语言编写的项目代码与相关资源。仓库整体以 Smalltalk 源代码为主，旨在保存、组织并共享该项目的类、包与示例。

## 主要内容
- src/ 或 packages/：Smalltalk 源代码（类与方法）。
- docs/：项目文档（若存在）。
- examples/：示例程序（若存在）。

> 本仓库的语言：Smalltalk（100%）

## 运行环境与依赖
- 需要一个 Smalltalk 映像（例如 Pharo、Squeak 或其他兼容 Smalltalk 的实现）。
- 建议使用支持 Monticello 或 Metacello 的工具来加载和管理包。

## 如何使用
1. 在本机安装并打开一个 Smalltalk 映像（推荐 Pharo）。
2. 使用 Monticello/Metacello 将仓库中的代码加载到映像中，或按需要将 .st / package 文件导入。
3. 在映像内浏览类与方法，运行示例或添加测试用例以验证功能。
