# 第1章：ST语言简介与环境配置

## 1.1 什么是ST语言？

ST（Structured Text，结构化文本）是一种高级文本编程语言，最初定义于IEC 61131-3国际标准。该标准是工业自动化领域最重要的编程规范之一，被全球绝大多数PLC厂商所支持。

### ST语言的起源与背景

```
IEC 61131-3 标准体系
├── IL (指令列表)        - 低级语言，类似于汇编
├── ST (结构化文本)      - 高级语言，本教程主题
├── FBD (功能块图)       - 图形化编程
├── LD (梯形图)          - 电气工程师常用
└── SFC (顺序功能图)     - 状态机编程
```

### ST语言的核心特点

| 特点 | 说明 |
|------|------|
| **高级语言语法** | 类似Pascal/C的结构化语法 |
| **强类型系统** | 编译时类型检查，减少运行时错误 |
| **结构化编程** | 支持函数、功能块封装 |
| **可移植性** | 符合标准可在不同厂商PLC间移植 |
| **数学能力强** | 适合复杂运算和算法实现 |

### ST vs 其他PLC语言

```st
// ST语言：计算斐波那契数列第n项
FUNCTION Fibonacci : INT
VAR_INPUT
    n : INT;
END_VAR
VAR
    i : INT;
    a, b : INT;
END_VAR

IF n <= 1 THEN
    Fibonacci := n;
ELSE
    a := 0; b := 1;
    FOR i := 2 TO n DO
        Fibonacci := a + b;
        a := b;
        b := Fibonacci;
    END_FOR
END_IF
```

## 1.2 ST语言的典型应用场景

### 何时选择ST语言

- ✅ 复杂数学运算（PID控制、坐标变换）
- ✅ 条件逻辑复杂的情况
- ✅ 需要算法封装和复用
- ✅ 从高级语言移植算法
- ✅ 数据处理和字符串操作

### 何时选择其他语言

- ⚠️ 简单的继电器逻辑 → 使用LD
- ⚠️ 顺序控制流程 → 使用SFC
- ⚠️ 简单的布尔逻辑 → 使用FBD/IL

## 1.3 开发环境配置

### 方案一：CODESYS（推荐）

CODESYS是最流行的IEC 61131-3开发环境，支持ST语言完整特性。

**下载地址**：https://www.codesys.com/

**安装步骤**：
1. 下载CODESYS Development System
2. 安装时选择完整安装（含示例库）
3. 安装目标设备驱动（如需要）

**创建新项目**：
```
文件 → 新建项目 → 选择设备 → 选择PLC编程语言为ST
```

### 方案二：TwinCAT 3（Beckhoff）

如果你使用Beckhoff硬件，TwinCAT是专业级选择。

**特点**：
- 集成Visual Studio开发环境
- 支持C++混合编程
- 强大的实时性能

### 方案三：在线仿真器（快速学习）

对于初学者，可以使用网页版仿真器进行练习：
- PLC-Lab
- PLCsim（西门子，仅限西门子硬件）

## 1.4 第一个ST程序

### 项目结构

```
MyProject/
├── Program/
│   ├── POU1.st          (* 主程序 *)
│   └── POU2.st          (* 其他程序 *)
├── Function/
│   └── MyFunctions.st   (* 自定义函数 *)
├── FunctionBlock/
│   └── MyBlocks.st      (* 功能块 *)
├── GlobalVariables/
│   └── GlobalVarList.gvl (* 全局变量 *)
└── DataTypes/
    └── MyTypes.st       (* 自定义类型 *)
```

### 经典入门程序：闪烁灯控制

```st
(* ============================================
 * 程序名称: BlinkingLight
 * 功能描述: 控制指示灯以1Hz频率闪烁
 * 作者: ST学习者
 * 日期: 2024
 * ============================================ *)

PROGRAM BlinkingLight
VAR
    (* 内部变量 *)
    timer : TON;              (* 通电延时定时器 *)
    blinkFlag : BOOL := FALSE; (* 闪烁状态标志 *)
    counter : INT := 0;       (* 计数器 *)
END_VAR

// 启动1秒定时器
timer(IN := NOT timer.Q, PT := T#500ms);

// 定时器到达时翻转状态
IF timer.Q THEN
    blinkFlag := NOT blinkFlag;
END_IF

// 输出到指示灯
Light := blinkFlag;

// 计数器累加
counter := counter + 1;

// 当计数器达到1000时复位
IF counter >= 1000 THEN
    counter := 0;
END_IF
```

## 1.5 语法基础规范

### 编码规则

| 规则 | 说明 | 示例 |
|------|------|------|
| 注释 | `(* ... *)` 或 `//` | `(* 这是注释 *)` |
| 语句终止 | 分号 `;` | `x := 5;` |
| 关键字 | 不区分大小写 | `BEGIN` = `begin` = `Begin` |
| 标识符 | 字母开头，可含数字和下划线 | `Motor1`, `temp_value` |

### 基本语法结构

```st
(* ============================================
 * 完整的POU (Program Organization Unit) 结构
 * ============================================ *)

(* 1. 程序/函数/功能块声明 *)
PROGRAM MyProgram
(* 或 FUNCTION MyFunction : INT *)
(* 或 FUNCTION_BLOCK MyBlock *)

(* 2. 变量声明区域 *)
VAR
    // 局部变量声明
    myVar : INT := 0;
    myArray : ARRAY[1..10] OF INT;
END_VAR

VAR_INPUT
    // 输入参数
    enable : BOOL;
    setpoint : INT;
END_VAR

VAR_OUTPUT
    // 输出参数
    status : BOOL;
    currentValue : INT;
END_VAR

VAR_CONSTANT
    // 常量
    MAX_VALUE : INT := 100;
END_VAR

(* 3. 执行代码区域 *)
BEGIN
    // 你的逻辑代码
    
    (* 条件判断 *)
    IF enable THEN
        myVar := setpoint;
        status := TRUE;
    ELSE
        myVar := 0;
        status := FALSE;
    END_IF
    
    currentValue := myVar;
    
(* 4. 结束标记 *)
END_PROGRAM
(* 或 END_FUNCTION *)
(* 或 END_FUNCTION_BLOCK *)
```

### 关键字速查表

| 类别 | 关键字 |
|------|--------|
| 程序组织 | `PROGRAM`, `FUNCTION`, `FUNCTION_BLOCK` |
| 变量声明 | `VAR`, `VAR_INPUT`, `VAR_OUTPUT`, `VAR_IN_OUT`, `VAR_CONSTANT`, `VAR_TEMP`, `VAR_GLOBAL` |
| 数据类型 | `INT`, `REAL`, `BOOL`, `STRING`, `TIME`, `DATE`, `ARRAY`, `STRUCT` |
| 控制结构 | `IF`, `THEN`, `ELSE`, `ELSIF`, `CASE`, `FOR`, `TO`, `BY`, `DO`, `WHILE`, `REPEAT`, `UNTIL`, `EXIT`, `RETURN` |
| 类型声明 | `TYPE`, `END_TYPE`, `ENUM`, `OF`, `STRUCT` |
| 其他 | `TRUE`, `FALSE`, `NULL`, `AND`, `OR`, `NOT`, `MOD`, `RET`, `VAR_CONFIG` |
